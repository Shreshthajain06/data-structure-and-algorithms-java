# DSA in Java Practice

This repository contains my practice solutions for **Data Structures and Algorithms (DSA) using Java**. The goal of this repository is to improve problem-solving skills and strengthen understanding of core DSA concepts commonly used in technical interviews and competitive programming.

## 📚 Topics Covered

* Arrays
* Strings
* Recursion
* Searching Algorithms
* Sorting Algorithms
* Pattern Problems
* Basic Problem Solving
* Logical Coding Challenges

## 🎯 Objective

The main objective of this repository is to:

* Practice Java programming
* Develop strong problem-solving skills
* Prepare for coding interviews and technical assessments
* Build a structured collection of DSA solutions

## 🛠 Technologies Used

* Java
* VS Code 
* Git & GitHub
